# 강릉 로컬 맵

강릉의 맛집, 관광, 주변상권 정보를 간단하게 둘러볼 수 있는 정적 웹사이트입니다.

## 실행 방법

1. 이 폴더에서 `index.html` 파일을 더블클릭합니다.
2. 브라우저가 열리면 메인페이지에서 원하는 서비스를 선택합니다.

별도의 설치나 서버 실행 없이 바로 사용할 수 있습니다.

## 파일 구조

- `index.html`: 메인페이지
- `food.html`: 맛집서비스 페이지
- `tour.html`: 관광서비스 페이지
- `business.html`: 주변상권서비스 페이지
- `popular-restaurants.html`: 인기맛집찾기 페이지
- `popular-cafes.html`: 인기카페찾기 페이지
- `popular-attractions.html`: 인기관광지 페이지
- `date-course.html`: 데이트코스 추천 페이지
- `nearby-stores.html`: 주변 상점찾기 페이지
- `local-currency.html`: 지역화폐 사용처 찾기 페이지
- `styles.css`: 전체 공통 디자인
- `script.js`: 카드 등장 애니메이션

## 참고

- 모든 페이지는 같은 폴더 안에 있어야 링크가 정상 동작합니다.
- 시작은 항상 `index.html`에서 하면 됩니다.
